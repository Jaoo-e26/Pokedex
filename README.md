📕 Pokédex – Front-End Interactive Landing Page

This application is an interactive Pokédex landing page, developed with a strong focus on front-end structure, responsiveness, and user experience. The project visually simulates a Pokédex device, using a layout inspired by a physical interface, with a centralized design and attention to visual details.

Pokémon data is locally managed, allowing dynamic rendering of images, numbers, and names without relying on external services. All navigation and search logic are handled entirely on the front end, ensuring offline functionality and full control over the data flow.

🛠️ Technologies Used:

- HTML5 for semantic markup and application structure

- CSS3 for advanced styling, responsiveness, and visual effects

- JavaScript (ES6+) for state management, DOM manipulation, and event handling

⚙️ Features:

* Dynamic rendering of locally stored Pokémon data

* Search by Pokémon name or number

* Sequential navigation between records (Previous / Next)

* Interface updates without page reloads

* Responsive layout adaptable to different screen sizes

🎯 Project Goal:

To demonstrate solid knowledge of front-end development fundamentals, including DOM manipulation, JavaScript logic, code organization, and the creation of interactive user interfaces, making this project suitable for a professional portfolio.
